# Installation

Follow these steps to set up the 0xHuman project locally.

## Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn
*   Git

## Steps

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/0xHuman.git
    cd 0xHuman
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in browser**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the application.
