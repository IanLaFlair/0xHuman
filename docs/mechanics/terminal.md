# Game Terminal

The Game Terminal provides a command-line interface (CLI) style interaction for advanced users and immersion.

## Features

*   **Command Input**: Type commands to navigate, check stats, or initiate actions.
*   **System Logs**: View battle logs and system notifications in a retro terminal format.
*   **Secrets**: Discover hidden commands and easter eggs.

## Common Commands
*   `help`: List available commands.
*   `status`: Show player status.
*   `clear`: Clear the terminal screen.
