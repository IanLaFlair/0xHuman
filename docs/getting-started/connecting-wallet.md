# Connecting Wallet

0xHuman uses **RainbowKit** for wallet management.

## How to Connect

1.  Click the **Connect Wallet** button in the top right corner of the navigation bar.
2.  Select your preferred wallet (MetaMask, Rainbow, Coinbase Wallet, etc.).
3.  Approve the connection request in your wallet extension or mobile app.
4.  Once connected, your address and avatar will be displayed.

## Supported Networks

Currently, the application supports the following networks:
*   Ethereum Mainnet
*   Polygon
*   Arbitrum
*   (Add others as configured in `wagmi.ts`)
