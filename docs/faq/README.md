# FREQUENTLY ASKED

### Q: Is the game provably fair?
**A:** Yes. We use on-chain randomness (VRF) to determine the matchmaking outcome.

### Q: Which wallet should I use?
**A:** Any EVM-compatible wallet connected to Mantle Network (Metamask, Rainbow, etc.).

### Q: What happens if the timer runs out?
**A:** If you fail to vote before 00:00, it counts as a forfeit (**Automatic Loss**).
